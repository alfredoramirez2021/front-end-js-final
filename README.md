# Gunners Diseños
📌 Descripción del Proyecto

Gunners Diseños es una página web desarrollada en HTML, CSS y Bootstrap 5 para mostrar productos personalizados como almanaques, toppers y stickers.
El sitio cuenta con un diseño moderno, responsive y organizado para ofrecer una experiencia visual atractiva a los usuarios.

El proyecto incluye:

Navegación principal
Carrusel de imágenes
Catálogo de productos
Sección de reseñas
Formulario de contacto
Carrito de compras
Diseño adaptable a dispositivos móviles
🖥️ Tecnologías Utilizadas
HTML5
CSS3
Bootstrap 5
SVG Icons
